Text Formatting Examples

This page illustrates the use of various text formatting tags in HTML.

Bold and Strong Text

The b element makes text bold without extra importance.

The strong element defines text with strong importance (usually displayed in bold).

Italic and Emphasized Text

The i element makes text italic (e.g. for technical terms).

The em element indicates emphasized text (displayed in italic).

Subscript and Superscript

This is subscript text (e.g. H2O).

This is superscript text (e.g. E = mc2).

Highlighted and Small Text

The mark element defines marked/highlighted text.

The small element defines smaller text (often used for copyright or fine print).

Deleted and Inserted Text

The del element defines text that has been deleted from a document.

The ins element defines a text that has been inserted into a document.

Preformatted Text

The pre element defines preformatted text. It preserves both spaces and line breaks:
