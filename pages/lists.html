HTML Lists

An HTML list is a collection of related items e.g., days of the week or ingredients in a recipe. This page will illustrate the various types of lists we can create in a webpage.

Ordered Lists

This is a collection of items in a particular order or steps that need to be followed or carried out in a given order.
