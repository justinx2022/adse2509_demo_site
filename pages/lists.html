HTML Lists

An HTML list is a collection of related items e.g., days of the week or ingredients in a recipe. This page will illustrate the various types of lists we can create in a webpage.

Ordered Lists

This is a collection of items in a particular order or steps that need to be followed or carried out in a given order.

Days in a Week:

  1. Sunday
  2. Monday
  3. Tuesday
  4. Wednesday
  5. Thursday
  6. Friday
  7. Saturday

Nested Lists

This is a list that contains other list(s) within it. An example is the list of courses:

ACCP/ADSE
  1. Semester 1
    1. Programming Practices & Techniques
    2. Elementary Programming In C
    3. Building Modernistic Website
    4. UI & UX for RWD
    5. Working with vectors
    6. Colour Theory
  2. Semester 2
    1. Video editing
    2. Sound editing
