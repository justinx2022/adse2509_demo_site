        <h1 class="page heading">Text Formatting Examples</h1>
        <p>This page illustrates the use of various text formatting tags in HTML.</p>

        <h3 class="section-heading">Bold and Strong Text</h3>
        <p>The <b>b element</b> makes text bold without extra importance.</p>
        <p>The <strong>strong element</strong> defines text with strong importance (usually displayed in bold).</p>

        <h3 class="section-heading">Italic and Emphasized Text</h3>
        <p>The <i>i element</i> makes text italic (e.g. for technical terms).</p>
        <p>The <em>em element</em> indicates emphasized text (displayed in italic).</p>

        <h3 class="section-heading">Subscript and Superscript</h3>
        <p>This is <sub>subscript</sub> text (e.g. H<sub>2</sub>O).</p>
        <p>This is <sup>superscript</sup> text (e.g. E = mc<sup>2</sup>).</p>

        <h3 class="section-heading">Highlighted and Small Text</h3>
        <p>The <mark>mark element</mark> defines marked/highlighted text.</p>
        <p>The <small>small element</small> defines smaller text (often used for copyright or fine print).</p>

        <h3 class="section-heading">Deleted and Inserted Text</h3>
        <p>The <del>del element</del> defines text that has been deleted from a document.</p>
        <p>The <ins>ins element</ins> defines a text that has been inserted into a document.</p>

        <h3 class="section-heading">Preformatted Text</h3>
        <p>The pre element defines preformatted text. It preserves both spaces and line breaks:</p>
        <pre>
    Text in a pre element
    is displayed in a fixed-width
    font, and it preserves
    both      spaces and

</pre>
